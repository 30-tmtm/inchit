# inchit 홈 화면 레이아웃 요소 정리 (MVP)

지금까지 확정된 방향(홈 단순화, 캐릭터 중심 히어로 카드, 체크리스트 미니카드 `보기` 진입, `주간 시간표`/`월간 캘린더` 탭 분리)을 기준으로 홈 화면 레이아웃 요소를 정리합니다.

## 1) 홈 화면 레이아웃 구조 (위에서 아래 순서)

### 1. 상단 앱바

- 좌측: 자녀 요약(이름, 월령)
- 우측: `검색`, `알림` 아이콘

### 2. 히어로 카드 (친밀감 핵심 영역)

- 상단: 육아 부모를 위한 1줄 메시지
  - 격려 메시지 또는 월령 기반 성장 발달 메시지
- 중앙: 아기 3D 캐릭터(시선 중앙 정렬)
- 하단: 아이 정보
  - 이름(기본 크기)
  - 개월수(기본 크기)
  - 나이(작게, 예: `만 1세`)
- 보조 원칙:
  - 메시지는 1~2줄(짧고 따뜻한 톤)
  - 의료적 단정 표현 금지(참고형 표현 사용)

### 3. 다음 일정 카드 (Next Event)

- 가장 가까운 일정 1건 강조
- 시간 / 제목 / 장소
- (선택) 준비물 1~2개 미리보기

### 4. 체크리스트 미니카드 (MVP 핵심)

- 제목: `오늘 체크리스트 (K-DST 기준)`
- 현재 월령 라벨
- 진행률: `2/5 완료`
- CTA: `보기` (홈에서는 토글하지 않음)

### 5. 오늘 주요 일정 요약 (선택)

- 최대 3개만 노출(예: 09:00, 11:30, 15:00)
- `전체보기` 클릭 시 `주간 시간표` 또는 `월간 캘린더` 탭으로 이동

### 6. 육아 포인트 카드 (선택)

- 건강검진/발달 정보 1~2개 요약
- `더보기` 버튼

## 2) 고정 네비게이션

- 하단 탭바(5개): `홈` / `주간 시간표` / `월간 캘린더` / `체크리스트` / `마이`

## 3) 홈 화면 상태별 표시 규칙

### 초기 빈 상태

- 일정이 없으면: "아직 일정이 없어요" + `시간표 만들기` CTA
- 체크리스트가 없으면: "월령 체크리스트 준비 중" + `체크리스트 보기`

### 일정이 많은 날

- 다음 일정 카드는 1개만 강조
- 체크리스트 미니카드는 고정 노출
- 나머지는 "오늘 주요 일정 요약(최대 3개)"으로 축약

### 알림이 많은 날

- 앱바 알림 아이콘에 배지 점만 표시
- 홈 본문 정보는 과밀하지 않게 유지

## 4) 권장 비율(모바일 기준)

- 상단 앱바: 10%
- 히어로 카드(메시지+캐릭터+아이정보): 34%
- 다음 일정 카드: 16%
- 체크리스트 미니카드: 14%
- 오늘 주요 일정 요약(선택): 16%
- 육아 포인트(선택): 10%
- 여백/스크롤 버퍼: 10%
